一些本站常用手法说明

  • 提示
    • 不推荐移动端阅读, 除非你有鼠标
    • 恰到好处的提示有助于帮助读者理解
    • 勿点击提示, 会回到文首
    • 隐藏提示能够简短篇幅
  • 链接
    • 无任何说明指向文章内部, 如一些简单的提示
    • 无明文说明指向站内文章, 如提示中’点击查看 XXX’ 等
    • 明确说明指向站外, 如文章中’点击跳转 XXX’等
