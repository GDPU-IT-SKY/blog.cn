VirtualBox 中安装 ArchLinux

本文字数： 322 阅读时长 ≈ 1 分钟

光盘映像来源

  • 官方网站
  • 各大开源软件镜像站

用于此实例虚拟机的额外配置

  • 操作系统版本 Linux (可选)
  • 处理器 2 个 (必需)
  • 内存 4096 MB (必需)
  • 启用 PAE (可选)
  • 视频控制器 VMSVGA (必需)
  • 视频内存 128 MB (可选)
  • 启用 3D 加速 (可选)
  • 储存控制器 SATA (必需)
  • 启用主机读写缓存 (可选)
  • 固态驱动器 (可选)

在未安装桌面环境时安装增强功能:

  1. 推荐
    1. 点击虚拟机菜单中的 安装增强功能
    2. 在虚拟机中 挂载 上一步插入的光盘映像到随意一个空目录下, 请忽略警告.
    3. 切换到挂载点, 开始 build. 如出现依赖问题, 请安装它提示的包. 一般 ArchLinux 安装 base-devel 包组即可.
    4. 重启 完成安装.
  2. 不推荐
